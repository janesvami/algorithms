# Algorithm Training Hub

A structured project for practicing and demonstrating various solutions to classic algorithm problems encountered in Java Junior interviews.

## Project Structure

Each problem is organized in its own package with:
- An interface defining the problem contract
- Multiple implementations showing different approaches

## Problems Covered

### Brackets
- **Stack + HashMap** – classic solution for validating parentheses

### Two Sum
- **Brute Force** – O(n²) time, O(1) memory
- **Two Pointers** – O(n) time, O(1) memory (requires sorted array)
- **HashMap** – O(n) time, O(n) memory

### FizzBuzz
- **If-Else** – O(n) time, O(1) memory
- **Switch** – O(n) time, O(1) memory
- **Stream API** – O(n) time, O(n) memory

### Palindrome
- **Inverted String** – O(n) time, O(n) memory
- **Two Pointers** – O(n) time, O(1) memory
- **With Asterisk** – O(n) time, O(1) memory (alphanumeric filtering + case insensitivity)

### Anagram
- **Sorting** – O(n log n) time, O(n) memory
- **Char Counting** – O(n) time, O(1) memory (Latin alphabet only)
- **HashMap** – O(n) time, O(n) memory

## How to Run

Run the `Main` class to see all solutions in action with predefined test cases.
